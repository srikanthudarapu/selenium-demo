Selenium Demo

This is a demo application created for demonstrating the basic and advanced features of the Selenium WebDriver. It also covers Selenium integration with TestNG (parallel cross-browser execution and reports) and Cucumber (BDD) frameworks. This application is created using Spring Boot, HTML, jQuery and Bootstrap.
